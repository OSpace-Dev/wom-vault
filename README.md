# 袋熊手账

袋熊手账是一款本地优先的个人资料整理与知识手账工具。

它适合把分散的记录、资料、备注和图片逐步整理成自己的长期资料库，重点是轻量、本地、可持续使用。

## 下载

请前往本仓库的 [Releases](../../releases) 页面下载最新版本。

当前首发版本提供：

- Windows 便携版

## 当前版本支持

- 工作台主视图
- 条目浏览与详情查看
- 条目创建与编辑
- 图片附件展示
- 本地备份导出与导入恢复

## 适合谁使用

- 想把个人资料集中整理在本地的人
- 希望长期积累自己的知识和记录的人
- 想用桌面工具管理文字、图片和资料的人

## 使用方式

1. 在 `Releases` 页面下载最新安装包
2. 双击运行
3. 按界面提示开始使用
